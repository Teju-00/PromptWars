# Civic Readiness Scoring Model

The India Election Companion Assistant evaluates voter preparedness using a structured readiness model.

### Score Components:

1. **Eligibility Awareness**
   - User understands minimum voting age (18+)

2. **Registration Awareness**
   - User knows how to apply using Form 6

3. **Document Preparedness**
   - User knows accepted identity documents

4. **Polling Knowledge**
   - User understands EVM + VVPAT workflow

5. **Constituency Awareness**
   - User knows polling location lookup process

6. **Election Timeline Awareness**
   - User understands election phases and deadlines

Each component contributes to the overall readiness score.

### Adaptive Guidance:
Gemini Assistant adapts guidance dynamically based on the readiness score level:

- **Low readiness**: Provides step-by-step onboarding guidance.
- **Medium readiness**: Offers document + polling preparation support.
- **High readiness**: Provides constituency lookup + simulation support.

This structured approach improves decision logic, assistant intelligence, and overall code quality.
