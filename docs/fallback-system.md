# Assistant Reliability Fallback System

The **India Election Companion Assistant** is designed for maximum reliability. If the Gemini API becomes unavailable or encounters a configuration error, the system activates a comprehensive fallback design.

### Fallback Resources:
The assistant provides manual fallback guidance including:
- **ECI Helpline Number**: 1950 (Voter Helpline).
- **Official NVSP Portal**: Direct links to `voters.eci.gov.in`.
- **Document Checklist References**: Static guides for age and residence proof.
- **Polling Preparation Steps**: Basic walkthroughs of the booth process.

### Benefits:
- **Security**: Ensures users are always directed to official, safe sources.
- **Reliability**: Provides mission-critical information even during service interruptions.
- **Production Readiness**: Demonstrates a robust design capable of serving as a real-world civic tool.
